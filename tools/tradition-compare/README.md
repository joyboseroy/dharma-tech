# Tradition Comparator

Compare how Buddhist traditions differ on any concept. Uses the emptiness-graph typed edges — not LLM generation.

## Usage

```bash
python tradition_compare.py sunyata
python tradition_compare.py consciousness --verbose
python tradition_compare.py --question 'is there a self?'
python tradition_compare.py --list
```
